
# ConsumerPokeApi

Aplicação Console em C# que consome a API pública PokeAPI.

## Funcionalidade

O programa:
1. Solicita o nome de um Pokémon.
2. Consulta a API https://pokeapi.co
3. Exibe no console:
- ID
- Nome
- Altura
- Peso

## Executar

No terminal dentro da pasta do projeto:

dotnet run

Exemplo:

Digite o nome do Pokémon: pikachu

=== Dados do Pokémon ===
ID: 25
Nome: pikachu
Altura: 4
Peso: 60
